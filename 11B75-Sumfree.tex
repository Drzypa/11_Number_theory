A set $A$ is called \emph{sum-free} if the equation $a_1+a_2=a_3$ does not have solutions in elements of $A$. Equivalently, a set is sum-free if it is disjoint from its $2$-fold sumset, i.e., $A \cap 2A =\emptyset$.
