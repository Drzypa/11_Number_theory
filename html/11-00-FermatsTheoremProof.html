Fermat’s theorem proof


Consider the sequence a, 2⁢a,…,(p-1)⁢a.

They are all different (modulo p), because if m⁢a=n⁢a with 1≤m<n≤p-1 then 0=a⁢(m-n), and since  p∤a, we get p∣(m-n),  which is impossible.

Now, since all these numbers are different, the set  {a, 2⁢a, 3⁢a,…,(p-1)⁢a}  will have the p-1 possible congruence classes (although not necessarily in the same order) and therefore

a⋅2⁢a⋅3⁢a⁢⋯⁢(p-1)⁢a≡(p-1)!⁢ap-1≡(p-1)!(modp)

and using  gcd⁡((p-1)!,p)=1  we get

ap-1≡1(modp).
