proof of properties of primitive roots


The material in the main article is conveniently recast in terms of the groups (ℤ/m⁢ℤ)×, the multiplicative group of units in ℤ/m⁢ℤ. Note that the order of this group is exactly ϕ⁢(m) where ϕ is the Euler phi function. Then saying that an integer g is a primitive root of m is equivalent to saying that the residue class of g(modm) generates (ℤ/m⁢ℤ)×.

Proof.

(of Theorem):
The proof of the theorem is an immediate consequence of the structure of (ℤ/m⁢ℤ)× as an abelian group; (ℤ/m⁢ℤ)× is cyclic precisely for m=2,4,pk, or 2⁢pk. ∎

Proof.

(of Proposition):

  1. 1.

    Restated, this says that if the residue class of g(modm) generates (ℤ/m⁢ℤ)×, then the set {1,g,g2,…,gϕ⁢(m)} is a complete set of representatives for (ℤ/m⁢ℤ)×; this is obvious.

  2. 2.

    Restated, this says that g(modm) generates (ℤ/m⁢ℤ)× if and only if g has exact order m, which is also obvious.

  3. 3.

    If g(modm) generates (ℤ/m⁢ℤ)×, then g has exact order ϕ⁢(m) and thus gs=gt(modm) if and only if gs-t=1 if and only if ϕ(m)∣s-t.

  4. 4.

    Suppose g generates (ℤ/m⁢ℤ)×. Then (gk)r=1 if and only if gk⁢r=1 if and only if ϕ(m)∣kr. Clearly we can choose r<ϕ⁢(m) if and only if gcd⁡(k,ϕ⁢(m))>1.

  5. 5.

    This follows immediately from (4).∎
