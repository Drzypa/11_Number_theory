eventually coincide


Let A and B be two nonempty sets of integers. We say that A and B eventually coincide if there is an integer C such that n∈A if and only if n∈B for all n≥C. In this case, we write A∼B, noting that the relation of eventually coinciding is clearly an equivalence relation.
