iterated sum of divisors function


The iterated sum of divisors function σk⁢(n) is ak in the recurrence relation a0=n and ai=σ⁢(ai-1) for i>0, where σ⁢(x) is the sum of divisors function.

Since n itself is included in the set of its divisors, the sequence generated by repeated iterations is an increasing sequence (that is, in ascending order). For example, iterating the sum of divisors function for n=2 gives the sequence 2, 3, 4, 7, 8, 15, etc. Erdős conjectured that there is a limit for (σk⁢(n))1k as k approaches infinity.
