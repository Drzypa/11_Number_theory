Dirichlet character


A Dirichlet character modulo m is a group homomorphism from (ℤm⁢ℤ)* to ℂ*. Dirichlet characters are usually denoted by the Greek letter χ. The function

γ⁢(n)={χ⁢(nmodm),if ⁢gcd⁡(n,m)=1,0,if ⁢gcd⁡(n,m)>1.

is also referred to as a Dirichlet character. The Dirichlet characters modulo m form a group if one defines χ⁢χ′ to be the function which takes a∈(ℤm⁢ℤ)* to χ⁢(a)⁢χ′⁢(a). It turns out that this resulting group is isomorphic to (ℤm⁢ℤ)*. The trivial character is given by χ⁢(a)=1 for all a∈(ℤm⁢ℤ)*, and it acts as the identity element for the group. A character χ modulo m is said to be induced by a character χ′ modulo m′ if m′∣m and χ⁢(n)=χ′⁢(nmodm′). A character which is not induced by any other character is called primitive. If χ is non-primitive, the gcd of all such m′ is called the conductor of χ.

Examples:

  • •

    Legendre symbol (np) is a Dirichlet character modulo p for any odd prime p. More generally, Jacobi symbol (nm) is a Dirichlet character modulo m.

  • •

    The character modulo 4 given by χ⁢(1)=1 and χ⁢(3)=-1 is a primitive character modulo 4. The only other character modulo 4 is the trivial character.
