multidimensional arithmetic progression


An n-dimensional arithmetic progresssion is a set of the form

Q =Q⁢(a;q1,…,qn;l1,…,ln)
={a+x1⁢q1+⋯+xn⁢qn∣0≤xi<li⁢ for ⁢i=1,…,n}.

The length of the progression is defined as l1⁢⋯⁢ln. The progression is proper if |Q|=l1⁢⋯⁢ln.
