perfect code


Let C be a linear (http://planetmath.org/LinearCode) (n,k,d)-code over 𝔽q.

The packing radius of C is defined to be the value

ρ⁢(C)=d-12.

The covering radius of C is

r⁢(C)=maxx⁡minc⁡δ⁢(x,c)

with x∈𝔽qn and c∈C, and where δ denotes the Hamming distance on 𝔽qn.

The code (http://planetmath.org/Code) C is said to be perfect if r⁢(C)=ρ⁢(C).

The list of of linear perfect codes is very short, including only trivial codes, Hamming codes (i.e. ρ=1), and the binary and ternary Golay (http://planetmath.org/BinaryGolayCode) codes.
