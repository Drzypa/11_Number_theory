Fraenkel’s partition theorem


Fraenkel’s partition theorem is a generalization of Beatty’s Theorem. Set

ℬ⁢(α,α′):=(⌊n-α′α⌋)n=1∞.

We say that two sequences partition ℕ={1,2,3,…} if the sequences are disjoint and their union is ℕ.

Fraenkel’s Partition Theorem: The sequences B⁢(α,α′) and B⁢(β,β′) partition N if and only if the following five conditions are satisfied.

  1. 1.

    0<α<1.

  2. 2.

    α+β=1.

  3. 3.

    0≤α+α′≤1.

  4. 4.

    If α is irrational, then α′+β′=0 and k⁢α+α′∉ℤ for 2≤k∈ℕ.

  5. 5.

    If α is rational (say q∈ℕ is minimal with q⁢α∈ℕ), then 1q≤α+α′ and ⌈q⁢α′⌉+⌈q⁢β′⌉=1.
