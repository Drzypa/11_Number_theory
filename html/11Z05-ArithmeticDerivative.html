arithmetic derivative


The arithmetic derivative n′ of a natural number n is defined by the following rules:

  • •

    p′=1 for any prime p.

  • •

    (a⁢b)′=a′⁢b+a⁢b′ for any a,b∈ℕ (Leibniz rule).

To define the arithmetic derivative of a negative number, we first note that 1′=0 by the Leibniz rule (1′=(1⋅1)′=1⋅1′+1′⋅1=2⋅1′, so 1′=0), and further that we must have

0=1′=((-1)⋅(-1))′=-2⋅(-1)′,

so (-1)′=0. The product rule now requires that we define (-n)′=-(n′)+n⁢(-1)′=-(n′).

Further, we can extend this definition to rational numbers by insisting that the quotient rule holds, i.e. for a prime p we should have

0=1′=(p⋅1p)′=(1p)′⁢p+1p,

giving us that

(1p)′=-1p2,

i.e. the usual quotient rule from calculus. We now complete the definition by extending multiplicatively (i.e. using the Leibniz rule).

The arithmetic derivatives for the first few positive integers are 0, 1, 1, 4, 1, 5, 1, 12, 6, 7, 1, 16, 1, 9, 8, 32, 1, 21, 1, 24, 10, 13, 1, 44, 10, etc.

As a consequence of p′=1 for a prime p, the arithmetic derivative of a semiprime (whether squarefree or not) works out to (p⁢q)′=p′⁢q+p⁢q′=1⁢p+q⁢1=p+q. For example, the arithmetic derivative of 10 is 7, which is 2 plus 5.

The only cases of n′=n for -1<n<1024 are 0, 4, 27.
