Sophie Germain prime


A prime number p is called Sophie Germain prime if 2⁢p+1 is also prime.

The first few Sophie Germain primes are: 2,3,5,11,23,29,41,53,83,89,113,131,173,179,191,233,…

It is conjectured that there are infinitely many Sophie Germain primes, but (like the Twin Prime Conjecture) this has not been proven. A heuristic estimate for the number of Sophie Germain primes less than n is 2⁢c⁢nln2⁡n, where c is the twin prime constant.
