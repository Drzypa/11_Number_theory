class number formula


Let K be a number field with [K:ℚ]=n=r1+2r2, where r1 denotes the number of real embeddings of K, and 2⁢r2 is the number of complex embeddings of K. Let

ζK⁢(s)

be the Dedekind zeta function of K. Also define the following invariants:

  1. 1.

    hK is the class number, the number of elements in the ideal class group of K.

  2. 2.

    RegK is the regulator of K.

  3. 3.

    ωK is the number of roots of unity contained in K.

  4. 4.

    DK is the discriminant of the extension K/ℚ.

Then:

Theorem 1 (Class Number Formula).

The Dedekind zeta function of K, ζK⁢(s) converges absolutely for ℜ⁡(s)>1 and extends to a meromorphic function defined for ℜ⁡(s)>1-1n with only one simple pole at s=1. Moreover:

lims→1⁡(s-1)⁢ζK⁢(s)=2r1⋅(2⁢π)r2⋅hK⋅RegKωK⋅∣DK∣

Note: This is the most general “class number formula”. In particular cases, for example when K is a cyclotomic extension of ℚ, there are particular and more refined class number formulas.
